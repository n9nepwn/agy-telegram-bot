# 🤖 AGY Telegram Bot

> 🎨 **100% Vibecoded** — This project was entirely generated by AI using [Google Antigravity](https://antigravity.google). Not a single line written by hand.

> Bridge between **Telegram** and **Google Antigravity** (AGY) — chat with AGY from your phone.

## 🚀 Features

- 💬 **Chat with AGY** directly from Telegram
- 🔄 **Streaming** — responses appear progressively as they're generated
- 🤖 **Multi-model** — switch between Gemini, Claude, etc.
- 📊 **Stats & quotas** — track your usage
- 🗂 **History** — conversations persisted in SQLite
- 🔐 **Whitelist** — only authorized Telegram User IDs can access
- ♻️ **Sessions** — independent context per user
- 🔧 **Full CLI** — `agy-telegram-bot` to manage everything from the terminal

## 📋 Prerequisites

- Python 3.11+
- A Google account with AGY (Google AI Pro plan recommended)
- AGY CLI installed and authenticated on the machine (`gcloud auth` or active AGY session)
- A Telegram bot created via [@BotFather](https://t.me/BotFather)

## ⚡ Quick Start (3 commands)

```bash
git clone https://github.com/n9nepwn/agy-telegram-bot.git && cd agy-telegram-bot
./install.sh
agy-telegram-bot start
```

The installer does everything automatically:
1. ✅ Creates the virtual environment
2. ✅ Installs all dependencies
3. ✅ Registers the `agy-telegram-bot` command globally
4. ✅ Launches the configuration wizard

## 🔧 CLI — Terminal Commands

```
agy-telegram-bot setup               Configure the bot (token, users, model)
agy-telegram-bot start               Start the bot in the background
agy-telegram-bot stop                Stop the bot
agy-telegram-bot restart             Restart the bot
agy-telegram-bot status              Bot status (PID, uptime, memory, config)
agy-telegram-bot logs                Live log tail (Ctrl+C to stop)
agy-telegram-bot logs -n 50          Last 50 lines of logs
agy-telegram-bot logs --clear        Clear the log file
agy-telegram-bot auth                List authorized users
agy-telegram-bot auth add <ID>       Authorize a Telegram user
agy-telegram-bot auth remove <ID>    Remove a user
agy-telegram-bot auth test           Test configuration (token, SDK, auth)
```

## ⚙️ Configuration

The `agy-telegram-bot setup` wizard configures everything interactively. Alternatively, edit `.env`:

| Variable | Description | Required |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | Bot token (from @BotFather) | ✅ |
| `ALLOWED_USER_IDS` | Authorized Telegram IDs (comma-separated) | ✅ |
| `DEFAULT_MODEL` | Default model | ❌ |
| `LOG_LEVEL` | Log level (DEBUG/INFO/WARNING/ERROR) | ❌ |
| `ENABLE_STREAMING` | Streaming responses (true/false) | ❌ |
| `SESSION_TIMEOUT_MINUTES` | Session inactivity timeout | ❌ |
| `MAX_CONTEXT_MESSAGES` | Max messages in context | ❌ |

> 💡 **Find your Telegram User ID**: send a message to [@userinfobot](https://t.me/userinfobot)

## 📝 Notes

- **Quotas**: The SDK uses the same quotas as AGY CLI/Desktop (Google AI Pro plan ~$20/mo)
- **No `vertex=True`**: default mode uses your personal Google account
- **Security**: NEVER share your `.env` — it contains your bot token
- **Sessions**: Each user gets their own AGY session with independent context
- **Auto-cleanup**: Inactive sessions are automatically closed
